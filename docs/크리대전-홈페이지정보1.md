# 디자인 가이드

- 현재 프로그램, 라인업, 참가기업, 크리에이터 정보는 미확정 상태입니다.
- 디자인은 **작년 행사 데이터를 임시 콘텐츠(더미 데이터)** 로 활용하여 제작 요청드립니다.
- 실제 콘텐츠는 추후 확정 후 교체 예정입니다.
- 콘텐츠 변경이 쉽도록 **레이아웃 중심**으로 제작 부탁드립니다.
- 기업 및 크리에이터는 카드(Grid) 형태로 구성하여 데이터만 교체할 수 있도록 설계합니다.
- 프로그램은 일정만 변경 가능한 구조로 제작합니다.
- 메인 비주얼 및 팝업은 관리자에서 이미지 교체가 가능하도록 구성합니다.

# 확인 필요 사항

## 휴대폰 인증 방식

현재 전달받은 내용이 상충되어 최종 확인이 필요합니다.

### ① 문자(PASS) 인증

- 구축비 : 30만원 (VAT 별도)

### ② 카카오 알림톡 인증

- 구축비 : 20만원 (VAT 별도)
- 알림톡 발송비 : 건당 10원
- 카카오 비즈니스 채널 필요

### 확인 요청

- 최종 인증 방식
- 구축 비용 부담 주체
- 알림톡 발송 비용 부담 주체

# 참고사항

- 단발성 행사 홈페이지입니다.
- 행사 종료 후 지속적인 운영 및 유지보수는 필요하지 않습니다.
- 관리자 페이지는 최소한의 콘텐츠 수정 및 등록 기능만 제공하면 됩니다.
- 라인업 및 프로그램 확정 후 데이터만 교체하여 오픈 예정입니다.
